# DemoLoop Integration

After completing each sprint, task set, or significant batch of work, write a `demo-session.md`
file in the project root. This file is used by DemoLoop to generate a voice demo walkthrough
for review.

## Format

```markdown
# DemoLoop Session

> Auto-generated by agent — edit before running `demoloop start`

## Sprint Summary
[2–3 sentences: what was accomplished and why it matters]

## Stories

### [Story or task title]
**Status:** Complete | In progress | Blocked
**What was built:** [What was implemented — be specific. Explain the behavior, not just the code.]
**What to look at:** [Exact UI elements, routes, API endpoints, or behaviors to demo. Say "scroll to...", "click...", "navigate to /path".]
**Files changed:** [comma-separated list of key files]
**Notes:** [Caveats, known issues, follow-ups, or things that need a decision]
```

## Rules

- Write one `### Story` block per logical feature or task — not per commit
- "What to look at" is the most important field — be specific enough that someone can follow along in the browser without reading code
- If something is incomplete or has a known issue, say so in Notes — don't hide it
- Keep each story under 100 words total
- Don't repeat the story title verbatim in the description — expand on it

## Example

```markdown
# DemoLoop Session

## Sprint Summary
Added user authentication and the first version of the settings page. Auth uses magic links so there's no password management. Settings page is minimal for now — just email preferences.

## Stories

### User authentication with magic link
**Status:** Complete
**What was built:** Email-based magic link auth via NextAuth. No passwords — users get a one-time link that signs them in for 7 days.
**What to look at:** Go to /login, enter any email, and check the network tab — you'll see the POST to /api/auth/magic-link. The session persists in an httpOnly cookie.
**Files changed:** src/auth/magic-link.ts, app/login/page.tsx
**Notes:** Rate limiting not yet implemented on the email endpoint.
```
